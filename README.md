# FE Test - PT Eigen Tri Mathema
By: Najim Rizky

## How to run project
1. Clone this repository
2. Open terminal on root directory
3. Run `cd fe-test-eigen`
4. Run `npm install`
5. Run `npm run dev`
6. Open `http://localhost:5173/` in your browser

## How to test
1. Clone this repository (if you haven't)
2. Open terminal on root directory
3. Run `cd fe-test-eigen`
4. Run `npm install`
5. Run `npm run test`
